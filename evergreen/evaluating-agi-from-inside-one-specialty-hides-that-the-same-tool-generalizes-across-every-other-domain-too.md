# Evaluating AGI from inside one specialty hides that the same tool generalizes across every other domain too

Source: [[thought-session-what-the-g-in-agi-actually-means|Thought Session: What the G in AGI Actually Means]]

The most common reaction from accomplished people in well-defined fields — biochemists, software engineers, financial analysts, authors — to AI is that it isn't quite good enough to do their work end-to-end. That assessment is empirically correct in narrow terms: language models complete 60 to 90 percent of any specialty task with varying remainder, and the remainder almost always requires human judgment. From inside a specialty, this looks like a tool that fails at the bar that matters.

What the specialty evaluation cannot see is that the same tool does the same 60-to-90-percent job across every other domain too. The financial analyst is using the very thing that, for the engineer, generates respectable code; for the biochemist, drafts experimental protocols; for the author, produces editorial feedback. The cross-domain reach is invisible to anyone who only tests the tool on one specialty, because the test never extends to where the novelty actually lives.

This is what makes the AGI claim feel strange to specialists. Prior paradigm shifts replaced one specialized tool with a better specialized tool — a faster machine, a more powerful framework, a more flexible spreadsheet. AGI is the first paradigm shift in which the *skeptic's specialty itself* is the thing the new tool generalizes across. The from-inside-the-specialty view is structurally unable to see what's new, because the novelty is precisely the spanning of every specialty rather than excellence within any one of them.

**Open questions:**
- Does this specialty-bound underestimation eventually correct itself as people see colleagues in other fields use the same tool, or does the specialty boundary remain cognitively binding even when external evidence is available?
- Is there a depth-of-specialty threshold past which the blindness becomes more or less acute? Hyper-specialists might be more blind because their specialty is more all-consuming, or less blind because deep specialists have lived through more paradigm shifts.
